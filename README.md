# Patient Health Database

Author: Jackson Kruger

## Introduction
This repository contains code for running a Patient Health Database.

This is written in python.

## Specifications
This datebase will contain:
* Patient Name
* Patient Age
* Patient Bloodtype
	- A+
	- A-
	- B+
	- B-
	- AB+
	- AB-
